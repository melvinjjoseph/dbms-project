## Getting Started
1. Clone the repository
2. Create a virtual environment (not needed on linux systems)
   ```bash
   python -m venv venv
   ```
   Activate the virtual environment
   ```bash
   venv\Scripts\activate
   ```
3. Install the dependencies
   ```bash
   pip install -r requirements.txt
   ```
4. Edit the database credentials in `database.py`
4. Run the app
   ```bash
   streamlit run main.py
   ```
